= image:./small-arc42-logo.png[arc42] Generator for Template Formats

If you want to use an arc42 template for your system, you **don't need to use this repo** -
just use one of the templates you find on http://arc42.org/download.

== General Overview

The content (structure + embedded-help) of the arc42 template are maintained
in the http://github.com/arc42/arc42-template repository. We call that one
the **Golden-Master**


== How to use this Project



This repository enthält ein Groovy-basiertes Build-System, das mit Standalone-Groovy-Skripten und dem Shell-Skript `build-arc42.sh` arbeitet. Gradle wird nicht mehr benötigt.

**Empfohlener Weg:**

1. Klone dieses Repository inkl. Submodule:

    git clone --recurse-submodules git@github.com:arc42/arc42-generator.git

2. Starte den Build-Prozess (installiert automatisch Pandoc, aktualisiert Submodule, baut alles):

    ./build-arc42.sh

    Die Ergebnisse findest du im `/build`-Verzeichnis.


== Build im Docker-Container

Zum Bauen ohne lokale Installation von Java, Groovy oder Pandoc:

    docker-compose up

Das aktuelle Verzeichnis wird als Volume eingebunden. Die Ergebnisse landen im lokalen `build/`-Ordner.

**Oder manuell:**

1. Submodule ggf. initialisieren:
    cd arc42-generator
    git submodule update --init --recursive

2. Golden Master aktualisieren (optional):
    cd arc42-template
    git checkout master
    git pull
    cd ..

3. Templates generieren:
    groovy build.groovy templates

4. Formate konvertieren:
    groovy build.groovy convert

5. Distribution erzeugen:
    groovy build.groovy distribution

**Shortcut:**
    groovy build.groovy

**Hinweis:** Das Skript `build-arc42.sh` erledigt alle Schritte automatisch und installiert Pandoc (Version 3.7.0.2) falls nötig.

Die Templates werden in verschiedenen Sprachen unter `build/src_gen` im AsciiDoc-Format generiert. Das System erkennt verfügbare Sprachen automatisch. Nach erfolgreichem Build können die ZIPs im Template-Repo committet werden.



=== Systemanforderungen

* http://groovy-lang.org[Groovy] Version 4.0 oder höher (getestet mit 5.0.2)
    - Installation z. B. via https://sdkman.io[SDKMAN]: `sdk install groovy`
* Java Runtime (mindestens Version 11, getestet mit OpenJDK 21)
* http://pandoc.org[Pandoc] Version 3.0 oder höher (getestet mit 3.7.0.2)
    - Das Skript `build-arc42.sh` installiert Pandoc automatisch, falls nicht vorhanden

Falls Tools fehlen, bricht der Build mit einer Fehlermeldung ab.
